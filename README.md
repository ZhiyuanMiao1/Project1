## 学生首页课程 Tab 与导师排序机制

Mentory 支持导师在个人名片中自定义「可授课课程」，并在学生首页按课程方向（Tab）浏览导师。其核心逻辑如下：

- 导师侧：导师在名片里填写可授课课程（文本/列表形式）。系统会对这些课程文本进行标准化与去重，并为其生成语义表示（embedding），用于后续的相关性匹配与排序。
- 学生侧：学生在首页点击某个课程方向 Tab 后，会以该方向作为筛选条件发起请求；服务端根据「该方向」与「导师名片课程」之间的语义相关度对导师进行排序，优先展示更匹配该方向的导师。
- Tab 顺序：课程方向 Tab 的展示顺序支持个性化配置（可在设置中调整），不影响上述“点选某个方向后按相关度排序”的机制。
